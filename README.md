# Pife

## _Regras_:

### Condição de vitória:

O jogador deverá formar 3 trios
de mesmo valor, mas naipes
diferentes (ex: 5♥, 5♣, 5♠) → Toak ou
3 trios de uma sequência do mesmo naipe
(ex: Q♦ → K♦ → A♦ ou 9♣ → 10♣ → J♣) → Flush

OBS: É possível ganhar o jogo de
imediato se a mão inicial for
compatível com a condição de vitória.

### Preparações:

No começo do jogo é distribuido nove
cartas para cada jogador

OBS: Para jogos com mais de dois jogadores
é recomendado mais de um baralho

### Dinâmica:

A cada rodada o jogador pode pegar uma
carta do baralho ou, se existir, comprar
a carta descartada pelo adversário
na vez do mesmo. O jogador pode manter uma
a carta ou descartar se ela não
for necessária.

OBS: Só é possível comprar uma carta
por rodada.
